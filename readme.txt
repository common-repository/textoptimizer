== Installation ==

This section describes how to install the Textoptimizer plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/textoptimizer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
